浏览器

易考支持所有主流浏览器，不论是何种操作系统或设备，只要是支持HTML5标准的浏览器都可以正常使用。 为了更好的用户体验，我们建议使用如下浏览器：

电脑端（包括Mac）：

  1. Internet Explorer 9.0及以上版本
  2. Chrome浏览器24.0及以上版本
  3. Firefox（火狐浏览器）20.0及以上版本
  4. Safari5及以上版本
  5. Opera浏览器15及以上版本

IOS移动终端：

  1. Safari
  2. Chrome
  3. Firefox

安卓移动终端：

  1. Google Chrome
  2. Firefox

视频监控浏览器

注意：若考试开启视频监控功能或者包含录音题，则必须使用如下浏览器：

电脑端（包括Mac）：

  1. Google Chrome
  2. Firefox
  3. Opera

安卓移动设备：

  1. Google Chrome
  2. Firefox

iOS移动终端：不支持

注意： iOS设备（iPhone和iPad）目前不支持视频监控功能和录音题；

Internet Explorer的任何版本也不支持视频监考功能。

很多国产浏览器大部分情况下可以正常使用易考，但不一定支持视频监控功能和录音题。

我们推荐使用Chrome或者FireFox浏览器，并更新到最新版本，以便获得最佳的跨平台支持。
